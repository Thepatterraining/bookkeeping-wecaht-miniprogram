🟢 **低风险**（主要风险：第三方库引入、Canvas 兼容性）

### 最关键发现

- **影响**: 无法展示收支趋势数据，需求完全无法满足
- **改造**: 需新增 `line-chart` 组件，基于 ECharts 或 Canvas 2D API 实现
- **外部依赖**: 需引入 `echarts-for-weixin` 或 `wx-charts` 库

#### 2. 🟢 数据聚合逻辑不足 (P1 - 部分阻塞)
- **发现**: 现有统计页面仅支持按月聚合分类数据，不支持日/周维度
- **位置**: `pages/statistics/statistics.js` 第63-94行 `calculateStatistics()` 方法
- **影响**: 无法提供需求中"日/周/月"三种时间维度的数据
- **改造**: 需扩展数据聚合逻辑，支持多维度计算

#### 3. 🟢 图表配色已有参考 (P2 - 非阻塞)

- **位置**: `components/budget-card/index.js` 第53-58行

### 2.2 数据处理逻辑

#### 2.2.1 账单数据处理调用链

```
pages/statistics/statistics.js [页面层]
→ loadBills() [获取数据]
→ calculateStatistics() [计算统计]
→ 按月过滤 → 按类型汇总 → 分类统计
```

#### 2.2.2 现有聚合逻辑分析

**位置**: `pages/statistics/statistics.js` 第63-94行

```javascript
// 第63-94行
calculateStatistics() {
  const { bills, currentMonth } = this.data
  const monthBills = bills.filter(bill => bill.date.startsWith(currentMonth))

  // 计算总收支
  const totalExpense = monthBills
    .filter(bill => bill.type === 'expense')
    .reduce((sum, bill) => sum + bill.amount, 0)

  const totalIncome = monthBills
    .filter(bill => bill.type === 'income')
    .reduce((sum, bill) => sum + bill.amount, 0)

  // 计算分类统计
  const categoryMap = new Map()
  monthBills.forEach(bill => {
    const key = `${bill.type}-${bill.category}`
    const current = categoryMap.get(key) || 0
    categoryMap.set(key, current + bill.amount)
  })

  // ...
}
```

**分支分析**:
- 分支1: `bill.type === 'expense'` → 计算支出总额
- 分支2: `bill.type === 'income'` → 计算收入总额
- 分支3: `bill.date.startsWith(currentMonth)` → 按月过滤

**问题识别**:
1. ❌ 仅支持按月聚合，不支持日/周维度
2. ❌ 无时间序列数据生成逻辑
3. ❌ 无趋势数据缓存机制

## 6. 问题识别与风险评估

### 6.1 技术风险

| 风险项 | 等级 | 影响 | 缓解措施 |
|--------|------|------|----------|
| ECharts 库体积大 | 🟡 中 | 小程序包体积增加 | 使用按需引入，仅导入折线图组件 |

| 风险项 | 等级 | 影响 | 缓解措施 |
|--------|------|------|----------|
| 数据聚合逻辑复杂 | 🟢 低 | 开发周期延长 | 复用 dayjs 库简化日期计算 |
| 多实例数据隔离 | 🟢 低 | 数据串扰 | 组件内部维护独立状态 |

## 8. 改造建议

### 8.1 技术选型建议

**推荐方案**: 使用 ECharts for 微信小程序

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| ECharts for Weixin | 功能强大、配置灵活、动画效果好 | 包体积较大（~300KB） | ⭐⭐⭐⭐⭐ |
